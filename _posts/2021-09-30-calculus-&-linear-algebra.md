微积分和线性代数是大部分理科、工科专业的必修课，其中的知识是后来很多专业课要用到的。非常重要！非常重要！

本文目的：描述掌握这两门课的最低标准。

## 微积分

| 周课时                                     | 微积分I：6 微积分II：6                                    |
| 本人成绩                                   | 微积分I：73 微积分II：91                                   |
| 课程教材                                   | 黄卫华等 《微积分I(II)》科学出版社 2013                    |
| 个人建议参考教材                            | 张筑生 《数学分析新讲(I-III)》北京大学出版社 1990           |

1. 一元函数微分学

| 极限                                       | 对函数极限的：“ε-δ语言”留下印象即可                          |
| 连续函数                                   |  留下图像式印象即可                                         |
| 导数                                       | 会用导数的定义求解常见函数的导函数，对反函数的导数留下印象即可  |
| 微分                                       | 熟练掌握 L'Hosptal 法则 & Taylor 公式                       |

重要技能点：熟记常用函数导数公式、正确使用L'Hosptal 法则 & Taylor 公式（对于非正确使用，以后可能会有文章详述）。

有趣点：自然对数e是怎么来的？

作为考察重点但实际上没用的：利用“常用不等式”的复杂证明题、微分学中值定理中关于复杂构造的压轴大题。

2. 一元函数积分学（牛顿-莱布尼兹公式是灵魂）

| 定积分定义                                 | 请对此留下深刻印象                                          |
| 积分计算                                   | 熟练掌握简单换元 & 分部积分                                  |

重要技能点：熟练掌握简单换元 & 分部积分。

作为考察重点但实际上没用的：复杂换元。此后课程考试中的涉及的积分都很简单，部分作业中困难积分要么可以用mathematica求解，要么可以编写程序进行数值积分。

3. 多元函数微分学

| 极限                                       | 对定义留下正确印象，能判断多元函数是否有极限                  |
| 偏导                                       | 连续可微、可微、连续、偏导存在的关系，不掌握也可              |
| 极值                                       | 驻点 -> 极值点                                             |

知识链接：考察极值点 -> 海涩矩阵 -> 线性代数

4. 多元函数积分学

重要技能点：二重积分、三重积分、第一/二类曲线积分、第一/二类曲面积分、格林公式。其中柱坐标、球坐标换元十分重要。

后续课程中，绝大部分情况下我们求解的问题都是高度对称的，尤其是柱坐标、球坐标问题，绝大部分可以将积分变量分离开积分再相乘。

知识链接：雅可比矩阵 -> 几何意义 -> 线性代数 《数学分析新讲II》张筑生 第十三章 5.a注记

知识链接：场论初步 -> 电磁场 《数学分析新讲III》张筑生 第十七章 附录 正交曲线坐标系中的场论计算 
 - [高等数学_场论初步_柱坐标、球坐标下梯度、散度、旋度] (https://www.bilibili.com/video/BV137411t7oY?spm_id_from=333.999.0.0)

5. 数项级数

基本技能点：正项级数：比较法之极限形式、根式法、比值法、 Cauchy-Hadamard 公式。（不需要很熟练）

重要技能点：逐项积分、逐项求导。电磁学、固体物理等课程中可能偶尔涉及。

6. 常微分方程

基本技能点：求解套路。

## 线性代数

| 学习时间                                   | 线性代数(第一层次)：大一上                                 |

| 本人成绩                                   | 78                                                       |
| 课程教材                                   | 张运清等 《线性代数讲义》科学出版社 2013                    |
| 个人建议参考教材                            | 任广千等 《线性代数的几何意义》西安电子科技大学出版社 2015    |

1. 行列式

作为考察重点但实际上没用的：诡异行列式的计算。评价：行列式只要记住基本计算方法即可，任何复杂计算在以后的课程中不会用到，却占到期中考试巨大分值，实属不解。

2. 高斯消元法

重要技能点：行简化梯形矩阵、高斯消元法解线性方程组、齐次以及非齐次方程组解的结构。

3. 矩阵

考察中没有被强调但非常重要的：线性变换与矩阵的关系。固体物理晶体宏观对称性习题中，部分同学不明白变换操作的矩阵是如何写出的，这一节给出了通俗的解答，过目不忘。 
 - 《线性代数的几何意义》任广千等 5.5 矩阵与线性变换关系的几何意义

个人认为应该放在比较靠前位置学但放在比较靠后位置学的：基变换与坐标变换。

考察中没有被强调但有点意思的：矩阵的秩与线性变换中变换前后图形维度的关系。
 - 《线性代数的几何意义》任广千等 6.3 线性方程组的秩及解的关系的几何意义

知识链接：计算物理导论-最小二乘法
 - 《线性代数的几何意义》任广千等 6.7 超定方程组的最小二乘解的几何意义

4. 特征值和特征向量

重要技能点：求解特征值与特征向量，在量子力学考试中是十分重要的。几何意义很简单，可以参阅《线性代数的几何意义》相关章节，不赘述。

5. 相似矩阵

P*-1 A P = B

重要意义：相似矩阵A和B是同一个线性变换（在同一线性空间中）在两个不同基下的表示矩阵。而可逆矩阵 P 就是基变换矩阵。

相似变换的理想目标：找到与A相似的对角化矩阵，实质是寻找一个适当的坐标系，使得该变换对这个新的坐标系上的单位向量（或基向量）只做伸缩变换，不做旋转变换。
 - 《线性代数的几何意义》任广千等 5.9 矩阵相似的几何意义

6. 二次型

相关概念：合同变换与合同矩阵。在教学中，内积空间相关部分被省去，我认为极不合理。我们要理解度量矩阵的作用是协调内积在不同基下算出的内积不同。对二次型进行的是合同变换，目的便是在于使二次型的函数值保持不变！

重要概念：正定、半正定、惯性指数等。及其几何意义。

此部分内容详见
 - 《线性代数的几何意义》任广千等 7 二次型的几何意义

7. 正交变换

既是相似变换，又是合同变换，便是正交变换。结合两个变换的几何意义可知，正交变换是一种刚性变换。在理论力学、固体物理等课程中有涉及。
 - 《线性代数的几何意义》任广千等 5.13 矩阵的等价、相似与合同关系
